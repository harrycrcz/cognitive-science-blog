# Cognitive Science Blog

A Next.js blog application about cognitive science topics.

## Pages

- **Home** (`/`) - Welcome page
- **About** (`/about`) - Information about the blog
- **Blog** (`/blog`) - List of articles

## Technologies Used

- Next.js
- React
- Tailwind CSS

## How to Run

```bash
npm install
npm run dev
```

Then open http://localhost:3000
